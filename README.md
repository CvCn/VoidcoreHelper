
# 关于插件 Features：：
- 打开地下城手册，随时查看晦暗虚空核心的掉落列表
- 切换拾取专精或者地下城手册的难度，查看其他掉落
- 地下堡和梦魇狩猎在小地图按钮查看

# 支持的游戏版本 Supported WOW Versions：
- 正式服12.0.5

*如果发现某些功能不能用，那就是还没做完*
